# Password-Manager
PassMe | Password Manager Tool

**Clone git repo by**

```bash
git clone https://github.com/aniketchavan2211/Password-Manager.git
```

**Run simply by Typing**

```bash
python main.py
python3 main.py
python3 ./main.py
```

**Run directly**
```bash
chmod +x main.py # give executable permission
./main.py
```

### Passing arguments

- help menu

**short arguments**
```bash
python3 main.py -h
```

**long arguments**
```bash
python3 main.py --help
```
